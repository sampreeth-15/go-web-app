# Go Web App

A simple web application built using **Golang** and the `net/http` package.
This project demonstrates how to serve static HTML pages using Go’s built-in HTTP server.

## Features
- Static file serving
- Multiple HTML pages (Home, About, Courses, Contact)
- Clean and simple project structure
- Beginner-friendly Go project

## Project Structure
go-web-app
├── main.go
├── static
│ ├── home.html
│ ├── about.html
│ ├── courses.html
│ ├── contact.html
│ └── images
